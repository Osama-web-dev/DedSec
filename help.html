CTF Help & Guidelines

How to Play

Capture The Flag (CTF) competitions are cybersecurity challenges where you find hidden "flags" in various challenges.

  1. Browse Challenges: Go to the Challenges page and select a category
  2. Read Description: Each challenge has a description explaining what you need to do
  3. Download Files: Some challenges require downloading files to analyze
  4. Find the Flag: Solve the challenge to find a flag in the format: dedSEC{example_flag}
  5. Submit Flag: Enter the flag in the submission box to earn points

Challenge Categories

Cryptography

Decrypt messages, break ciphers, and solve encryption-based puzzles. Tools like CyberChef, Python cryptography libraries, and frequency analysis are useful.

Reverse Engineering

Analyze compiled programs to understand their functionality. Use tools like Ghidra, IDA Pro, Radare2, and debuggers.

Web Exploitation

Find and exploit vulnerabilities in web applications. Common techniques include SQL injection, XSS, CSRF, and directory traversal.

Forensics

Analyze digital artifacts, network traffic, images, and file metadata. Tools include Wireshark, Binwalk, Steghide, and various hex editors.

Binary Exploitation

Exploit vulnerabilities in binary programs. Skills include buffer overflows, format string vulnerabilities, and ROP chains.

Useful Tools

Python

Essential for scripting and solving various challenges

Tips & Strategies

  • Start with easier challenges to understand the CTF format
  • Read challenge descriptions carefully - they often contain important hints
  • Use hints wisely - try to solve on your own first
  • Collaborate and discuss with other participants (if allowed)
  • Keep notes of techniques that work for different challenge types
  • Practice regularly to improve your skills
  • Don't get discouraged - some challenges are meant to be difficult

Frequently Asked Questions

What is the flag format?

All flags follow the format: dedSEC{text_here}. Make sure to include the "dedSEC{}" wrapper when submitting.

Can I reset my progress?

Yes, you can reset your progress from the Dashboard or Profile page using the "Reset Progress" button.

Do I need to create an account?

While you can browse challenges without an account, you need to register and login to submit flags and track your progress.

Are there any time limits?

No, this is a self-paced learning platform. Take your time to understand and solve each challenge.
